# Online Job Bank DBMS

## Overview
This repository contains an **Online Job Bank Database Management System** developed using **Java** and **SQL**. The system models the backend of a job recruitment platform, enabling interaction between job applicants, recruiters, and companies. Core functionalities include job postings, job applications, resume management, interview scheduling, and application status tracking.

The backend is implemented using a **BCNF-normalized relational database**, ensuring data integrity, consistency, and minimal redundancy. The frontend consists of a **Java-based graphical user interface (GUI)** that allows users to interact with the database intuitively. This project was developed as part of a **Relational Database Systems** course and demonstrates key database concepts such as entity-relationship modeling, schema design, functional dependencies, normalization (3NF and BCNF), SQL querying, and relational algebra.

## Features
- Job search and filtering
- Job application submission and tracking
- Resume upload and management
- Interview scheduling
- Recruiter job posting and applicant review
- Company-level job and recruiter management
- Java GUI using Java Swing for database interaction
- SQL queries with relational algebra representations using Oracle 11g database
- Unix-based menu using Bash/Shell Scripts for database interaction

## Technologies Used
- Java Swing (GUI development)
- Oracle 11g Database for SQL (Relational database design and queries)
- Bash/Shell Script for Unix-based menu
- BCNF-normalized schema

This project reinforces foundational database principles, including ER modeling, schema refinement, normalization, and query formulation, while integrating a functional user interface to demonstrate real-world applicability.
